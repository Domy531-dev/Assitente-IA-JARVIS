# Jarvis 1.5 Flash

**Jarvis** é um assistente de IA conversacional que permite interação por **voz** e **texto**, utilizando Python e a API da OpenAI.

## 🚀 Novidades da versão 1.5 Flash
- Modularização do código (voz, IA e main separados).
- Configurações externas via `config.json`.
- Suporte a múltiplos modelos GPT.
- Controle de velocidade e volume da voz.
- Logs mais claros e mensagens de status.

## 📦 Requisitos
- Python 3.8+
- Chave de API da OpenAI
